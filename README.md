learning-lisp
=============
Repository created in order to learn basics of Lisp

## Installation

Ubuntu:

```shell
sudo apt-get install clisp
```

## Running Lisp program

Source of "Hello world" Lisp program:

```lisp
; hello world lisp program.
(print "Hello World")
```

Running Lisp program:

```
clisp hello.lisp
```
